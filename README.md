# AI Algorithm Learning Agent

这是一个面向大学生编程学习场景的 AI 学习辅助 Agent 原型项目，主要用于辅助整理 C++ 算法模板、解释代码逻辑、总结易错点，并生成复习计划。

## 项目背景

在学习 AcWing 算法基础课、C++ 算法模板、Java 基础和英语四级备考的过程中，我发现自己经常遇到以下问题：

- 算法视频看懂了，但离开答案后难以独立复现代码
- 快速排序、二分、前缀和等模板容易混淆
- 学习内容分散，后期复习不方便
- 代码中的边界问题和易错点不容易及时发现

因此，我尝试构建一个 AI 驱动的学习辅助 Agent，用来帮助我整理算法模板、分析代码逻辑、生成复习计划，并把学习过程沉淀到 GitHub 仓库中。

## 核心功能

- 算法知识点解释
- C++ 代码逐行分析
- 算法模板整理
- 易错点总结
- GitHub 仓库结构生成
- 学习计划辅助制定
